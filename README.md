### How to run program:
#### Run python code:
```bash
   # Load the database as a .db file
   python3 load_data.py
   # Run the main program
   python3 main.py
```
### Options for program:
1. query
2. delete
3. insert
4. update
5. help
6. exit
Then you are able to write the SQL requests for options 1-4
