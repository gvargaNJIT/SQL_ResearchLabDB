# Research Lab Database using mySQL
## by Nicholas Bayuelo, Grace Varga, and Maheen Siddiqui
### How to run program:
#### Create virtual environment:
```bash
   python3 -m venv venv
   source venv/bin/activate
```
#### Install requirements.txt:
```bash
   pip install -r requirements.txt
```
#### Run python code:
```bash
   python3 main.py
```
### Options for program:
1. Query
2. Delete
3. Insert
4. Update
